CrossCompare.js

JavaScript library for quick dc.js chart comparison

Quick chart comparison

CrossCompare is a JavaScript library for quickly comparing charts built using dc.js. Bundled with crossfilter, dc.js provides fantastic in-browser dimensional filtering and charting. CrossCompare now adds the ability to quickly compare different filter states in a separate chart.

Download v1.0

Advancing dashboarding

CrossCompare started as a Master of Science project for Software Development at the University of Glasgow. This project is meant to evaluate the marketablility of open-source, web-based dashboard tools and seeks to advance its status quo.
